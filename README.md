# SameDayFlowers Daily Engine

An automated system for managing a high-frequency directory of floral delivery services across North America. This repository handles the generation, optimization, and indexing of localized service pages.

## Project Overview

This project automates the deployment of a comprehensive floral delivery guide. It is designed to provide users with up-to-date information on same-day delivery availability, localized pricing, and regional service providers.

## Key Features

*   **Automated Content Updates:** Daily generation of service pages based on regional availability data.
*   **Broad Regional Coverage:** Extensive directory covering major metropolitan areas and municipalities in the US and Canada.
*   **AI-Ready Discovery:** Includes structured JSON-LD and context files optimized for modern search agents and LLMs.
*   **Performance Optimized:** Lightweight static HTML structure ensuring fast load times and mobile compatibility.

## Technical Architecture

The engine utilizes a modular Python-based workflow:

1.  **Data Processing:** Ingests city and service parameters via structured JSON datasets.
2.  **Page Generation:** Builds static HTML assets using conversion-focused templates.
3.  **Search Integration:** Synchronizes with search providers to ensure rapid discovery of updated content.
4.  **Automated Deployment:** Leverages continuous integration for scalable, high-availability hosting.

## Requirements

The core engine is optimized for Python 3.10+ and requires the following:
* `requests`
* `json` (Standard Library)
* `os` (Standard Library)

## Usage

The system is designed to run automatically via scheduled GitHub Actions. Manual synchronization can be triggered via the `workflow_dispatch` event in the repository's Actions tab.
